# Python Test Automation

# Requirements
- Python 3
- Requests
- Pytest

# How use

- Clone the project from https://github.com/Maycowpq225/PythonBackendAutomation
- Open the project in your IDE, such as Pycharm or VSCode
- Install requirements using the following command in your terminal ```$ pip install -r requirements.txt```
- Turn on your virtual environment using the following commands in your terminal ```$ python -m venv venv``` and then ```$ source venv/Scripts/activate```
- Finally to execute tests, run the following command on your terminal ```$ pytest```
